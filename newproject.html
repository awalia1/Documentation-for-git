Starting a new project

1. Create a new folder for your project

You can use the comand or $ mkdir (folder-name) or you can do this through the interface.

2. Initialize Git repository inside project.

Navigate to the porject folder inside of terminal and the run the following command
$git init

You should see the following output:
Initialized empty Git repository in /users(ParentFolder)/(project)/.git/

3. Open a browser and go to Github and create a new repository

Once you create a new repository, you will copy and paste the following commands into terminal. These commands set the remote to point to your project folder...

$ git remote add origin https://github.com/iexample/day5.git

As well as starting a new branch for you to work on. This branch should be called master.

$ git push -u origin master

You're all set!

Now read about adding files to your project and pushing your code to Github!
